步骤

借助云游君的科普性操作网站——教你如何从零开始搭建一个属于自己的网站 ，初步完成了网站半成品操作，先总结如下：

1、安装Node.js

Node.js | 百度百科

JavaScript | MDN

下载Node.js

npm install -g yarn

2、安装git

3、注册GitHub账号

4、安装hexo

5、使用hexo主题

6、生成静态文件

hexo clean

hexo generate # 缩写为 hexo g

7、与远程仓库建立关联

git init # 初始化 Git 仓库，只需要执行一次即可

git checkout -b hexo#新建分支

8、部署

npm install hexo-deployer-git#为了更方便的部署到 GitHub Pages，Hexo 提供了 hexo-deployer-git 插件

hexo deploy#部署

9、备份与自动部署

添加到缓存区

git add -A
git commit -m “这次做了什么更改，简单描述下即可”

推送至远程仓库

git push
#第一次提交，你可能需设置一下默认提交分支#git push --set-upstream origin hexo
